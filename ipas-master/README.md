# IPAS-Master（アイパス・マスター）

**ストラテジ・マネジメント・テクノロジ完全攻略プラットフォーム**

本番URL: <https://ipas-master.vercel.app>

## 目的

ITパスポート試験の広範なシラバスを、AIの伴走とインタラクティブな演習で効率的に学習する。単なる過去問演習ではなく、**「計算問題の解法アルゴリズム」** と **「用語の構造的理解」** をターゲットにし、文系初学者やつまずきやすい受験者が合格まで自走できる環境を提供する。

## 主要機能一覧

| 機能 | 内容 | 試験対策への直結要素 |
| --- | --- | --- |
| 問題ドリル（500問） | 100問×5バッチ、14分野別フィルタ、シャッフル、プログレスバー、完了画面 | 全分野の知識・計算問題対策 |
| 損益分岐点シミュレーター | 固定費・変動費率・売上高のスライダー操作、BEPグラフ描画（Recharts） | ストラテジ系の計算問題対策 |
| 2進数変換ツール | 8ビットトグル、2進/10進/16進/ASCII/グレースケール同時表示 | テクノロジ系（基数変換・情報量）対策 |
| 用語フラッシュ（45用語） | 3分野+サブカテゴリ、たとえ話（メタファー）付き解説、正解/不正解カウント | 紛らわしい専門用語の識別・定着 |
| 計算問題ドリル | 基数変換・論理演算・損益分岐点・稼働率・ネットワーク・EVM計算 | マネジメント・テクノロジ系の計算問題対策 |

### 問題バンク構成（500問）

| バッチ | 問題番号 | 主な分野 |
| --- | --- | --- |
| 第1回 | Q1〜100 | 企業活動・経営戦略・システム戦略・法務 |
| 第2回 | Q101〜200 | DX・AI活用・プロジェクトマネジメント・サービスマネジメント |
| 第3回 | Q201〜300 | システム監査・基礎理論・コンピュータシステム |
| 第4回 | Q301〜400 | ネットワーク・データベース・セキュリティ |
| 第5回 | Q401〜500 | セキュリティ(追加)・AI/データサイエンス・総合問題 |

出題モード: 100問単位 / 14分野別 / シャッフルON/OFF

## UD・アクセシビリティ対応

| Feature | Detail |
| --- | --- |
| フォント | BIZ UDPGothic優先（ユニバーサルデザイン書体） |

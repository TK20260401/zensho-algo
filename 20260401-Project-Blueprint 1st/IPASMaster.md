# IPAS-Master（アイパス・マスター）

**ストラテジ・マネジメント・テクノロジ完全攻略プラットフォーム**

本番URL: <https://ipas-master.vercel.app>

---

## 1. プロジェクト概要

**ITパスポート（Iパス）試験対策**に完全にフォーカスし、かつ「自走自律」を促すWebアプリ。

Iパスは範囲が広いため、単なる過去問演習ではなく、**「計算問題の解法アルゴリズム」** と **「用語の構造的理解」** をターゲットにする。16年間の教育実践（視覚化の徹底）をIT実装に落とし込み、文系初学者やつまずきやすい受験者が、合格まで自走できる環境を提供する。

## 2. 設計思想

| 設計原則 | 詳細 |
| --- | --- |
| **SDL（自己主導型学習）** | AIの伴走とインタラクティブな演習で「自ら手を動かして」理解するプロセスを重視 |
| **視覚化の徹底** | 数値をいじりながら結果がリアルタイムに変わるシミュレーターで、体験的理解を促進 |
| **ゲーミフィケーション** | 正答率・連続正解・プログレスバーによるモチベーション維持 |
| **ユニバーサルデザイン** | 全漢字ルビ、WCAG AA準拠、タッチ操作最適化で誰もが使える設計 |

### 実装済み機能の詳細

| 機能 | 内容 | 試験対策への直結要素 |
| --- | --- | --- |
| 問題ドリル（500問） | 100問×5バッチ、14分野別フィルタ、シャッフル、プログレスバー、完了画面 | 全分野の知識・計算問題対策 |
| 損益分岐点シミュレーター | 固定費・変動費率・売上高のスライダー操作、BEPグラフ描画 | ストラテジ系の計算問題対策 |
| 2進数変換ツール | 8ビットトグル、2進/10進/16進/ASCII/グレースケール同時表示 | テクノロジ系（基数変換・情報量）対策 |
| 用語フラッシュ | 45用語、メタファー付き解説、正解/不正解カウント | 紛らわしい専門用語の識別・定着 |
| 計算問題ドリル | 基数変換・論理演算・損益分岐点・稼働率・ネットワーク・EVM計算 | マネジメント・テクノロジ系の計算問題対策 |

### 学習コンテンツ例（視覚化の仕掛け）

- **テクノロジ系：2進数と情報量** — 0と1のスイッチを切り替えると、リアルタイムで10進数や16進数に変換され、色の変化や文字（ASCII）として表示される「デジタル表現視覚化ツール」
- **ストラテジ系：損益分岐点分析** — 固定費、変動費、売上高のグラフをドラッグして動かし、利益が出るポイント（損益分岐点）がどう変化するかを体感するシミュレーター
- **マネジメント系：ガントチャートとアローダイアグラム**（未実装） — タスクの期間を変えると、クリティカルパスが自動で赤く強調される工程管理シミュレーター

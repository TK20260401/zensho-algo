# Logic-Riichi (Education Edition)
### アルゴリズム思考を育む「自走型」麻雀学習プラットフォーム

本日（2026/04/01）のキックオフに基づき、16年間の教育知見とモダンなITスタックを融合させた「教育版麻雀」のプロトタイプ開発プロジェクトを開始します。

## 1. プロジェクト概要 (Project Description)
麻雀の複雑なルールを「データ構造」と「アルゴリズム」に分解し、AIによるリアルタイムナビゲーションを通じて、初心者が「情報のデジタル化」と「論理的思考」を自走して習得できる学習OSを目指します。

## 2. 目的 (Objective)
- **教育的価値**: 共通テスト（情報Ⅰ）やFE（基本情報）に必要な「アルゴリズム構築力」の育成。
- **挫折の解消**: 初心者がつまずく「役の暗記」や「点数計算」をシステムが自動補完し、成功体験を積ませる。
- **自走自律**: AIメンターによるコーチングにより、教員に依存しない自律的学習（SDL）を実現する。

## 3. 主要な機能一覧 (Key Features)
| 機能名 | 役割・メカニズム | 対応概念 |
| :--- | :--- | :--- |
| **Algorithm Visualizer** | 牌の整列（理牌）プロセスを視覚化し、内部のソート挙動をアニメーション表示。 | 整列（ソート）アルゴリズム |
| **Logic Analyzer** | 14枚の構成を「4面子＋1雀頭」に分解する全探索をバックグラウンドで実行。 | 再帰処理・多次元配列 |
| **Probabilistic Simulator** | テンパイ時の待ち牌数や、上がりへの期待値をリアルタイムで数値化。 | モデル化とシミュレーション |
| **AI Learning Scaffold** | 答えを教えず、論理的なヒントを生成して思考を促す。 | 自走自律学習 (SDL) |
| **Learning Trace DB** | 思考ログを蓄積し、個別の弱点分析レポートを自動生成。 | データサイエンス・統計分析 |

## 4. 技術スタック (Tech Stack) - ベータ版
- **Frontend**: Next.js (Vercel) / Tailwind CSS
- **Backend/DB**: Supabase (Auth / PostgreSQL)
- **AI/Dev Support**: Cursor / Claude 3.5 Sonnet
- **Infrastructure**: Vercel (CI/CD)
